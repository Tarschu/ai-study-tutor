# 电工电子学知识库

## 读图复核重点

- 电路源：确认电压源/电流源方向、正负极、交流/直流、有效值/最大值。
- 电阻网络：确认串并联关系，不要只凭视觉距离判断连接；优先按节点判断。
- 电流与电压参考方向：题目给定方向不一定等于真实方向，计算结果为负表示实际方向相反。
- 开关与暂态：确认开关动作前后状态、动作时刻、电容初始电压、电感初始电流。
- 半导体器件：确认二极管方向、稳压值、三极管类型 NPN/PNP、管脚与偏置。
- 数字逻辑：确认门电路符号、输入取反小圆点、真值表顺序和高低电平约定。

## 常用基础

- 欧姆定律：`$U = IR$`
- 功率：`$P = UI = I^2R = U^2/R$`
- 基尔霍夫电流定律：节点电流代数和为 0。
- 基尔霍夫电压定律：闭合回路电压代数和为 0。
- 电容：`$i_C = C \frac{du_C}{dt}$`，稳态直流时可视为开路。
- 电感：`$u_L = L \frac{di_L}{dt}$`，稳态直流时可视为短路。
- 一阶电路：`$x(t)=x(\infty)+[x(0^+)-x(\infty)]e^{-t/\tau}$`
- RC 时间常数：`$\tau = RC$`；RL 时间常数：`$\tau = L/R$`
- 正弦量有效值：`$U = U_m/\sqrt{2}$`，`$I = I_m/\sqrt{2}$`
- 阻抗：`$Z_R=R$`，`$Z_L=j\omega L$`，`$Z_C=1/(j\omega C)$`
- 复功率：`$\underline S = P + jQ = \underline U \underline I^*$`
- 功率因数：`$\cos\varphi = P/S$`

## 讲解策略

- 先把电路抽象成节点、支路、回路，再写方程。
- 对新手先解释“参考方向”和“实际方向”的区别。
- 对暂态题，固定按三步讲：求初值 `0^+`、求终值 `\infty`、求时间常数 `\tau`。
- 对交流题，先说明相量法把微分问题转为代数问题。
- 对电子电路题，先判断工作区：截止、放大、饱和或二极管导通/截止。

## 常见错因

- 把电压源正负极看反。
- 把串联和并联看错，尤其是跨接导线导致同一点电位相同。
- 忘记电容电压不能突变、电感电流不能突变。
- 混淆有效值和最大值。
- 计算功率因数时漏掉相位符号或感性/容性判断。
